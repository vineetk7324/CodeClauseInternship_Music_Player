# CodeClauseInternship_Music_Player

This is a basic music player built using Python. The player allows you to play, pause, stop, and navigate through a playlist of MP3 files. It uses the pygame library for audio playback and provides a simple interface for controlling the music playback.

Features
Play, pause, stop, next, and previous controls for music playback.
Playlist management to add and navigate through songs.
Uses the pygame library for audio playback.
Requirements
Python 3.x
Library: pygame
Installation
Clone or download the repository to your local machine.

Install the required library using the following command:
pip install pygame
Place your MP3 files in the same directory or specify their paths in the script.

Run the script:
python music_player.py

Usage
Run the script.
Use the provided buttons to control music playback (Play, Pause, Stop, Next, Previous).
To add songs to the playlist, modify the script to include the file paths.
To exit the program, close the window or press Ctrl + C.
